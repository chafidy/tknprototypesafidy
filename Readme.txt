# 🌐 Teknet Group – Prototype du Site Web Officiel

Ce dépôt contient le prototype du site web de **Teknet Group**, une entreprise spécialisée dans les solutions digitales sur mesure, l’intégration de systèmes, l’IoT et les services informatiques à Madagascar.

## 🔧 Technologies utilisées

- HTML5 / CSS3
- JavaScript Vanilla
- Bootstrap 5
- Swiper.js (pour les carrousels)
- AOS (Animate On Scroll)
- Icônes Bootstrap et personnalisées

## 🎯 Objectifs du site

- Présenter les services et les expertises de Teknet
- Mettre en avant les réalisations et les références clients
- Faciliter la prise de contact via un formulaire et des CTA stratégiques
- Optimisation pour une bonne expérience utilisateur (UX) sur tous les supports

## 🧩 Structure du prototype

- `index.html` : page d’accueil avec présentation, services, portfolio, témoignages, contact
- `assets/` : dossier contenant les images, les icônes et les fichiers CSS/JS personnalisés
- `css/` : style principal du site (`style.css`)
- `js/` : scripts JS pour l’animation et l’interactivité

## 📸 Fonctionnalités clés

- Carrousel client interactif (Swiper.js)
- Effets d’apparition animés (AOS)
- Design responsive et adaptatif
- Intégration future possible avec CMS ou backend (non inclus ici)
